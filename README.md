- [Styled Components](https://styled-components.com/)

## :computer: Project

This project is a Discord based UI clone

## :runner: How to run

Install dependencies
```
  $ yarn

  or

  $ npm install
```

Start server
```
  $ yarn start

  or

  $ npm start
```

To build the app
```
  $ yarn build
```
